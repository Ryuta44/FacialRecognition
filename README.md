# FacialRecognition
顔認証のサンプルコード

# Requirement
Python 3.10.9<br>
pip 22.3.1<br>
numpy 1.24.2<br>
opencv-python 4.7.0.72<br>
Pillow 9.4.0<br>
facenet-pytorch 2.5.2

# Usage
このリポジトリをクローン
```
git clone git@github.com:Ryuta44/FacialRecognition.git
```

プロジェクトディレクトリに移動
```
cd FacialRecognition
```

自身の顔写真をプロジェクトフォルダに"image4.jpg"で格納

コードを実行
```
python3 signup.py
python3 login.py
```
